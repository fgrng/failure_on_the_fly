# Erhebungs-Teilnahmen sind pseudonym und strikt vom Trainingsbereich getrennt

Der Zugang zu einer Erhebung läuft über einen **Teilnahme-Link**, der für alle Teilnehmenden einer **Stichprobe** identisch ist. Beim Öffnen entsteht ein **Teilnahme-Token**, das die Forschungsdaten dieser Teilnahme bündelt und im Browser hinterlegt wird, sodass eine unterbrochene Teilnahme fortgesetzt werden kann. Das Token ist **strikt getrennt** von jedem Nutzerkonto und von jeder Aktivität im Trainingsbereich — dieselbe Person, die im Training übt und an einer Erhebung teilnimmt, ist forschungsseitig nicht verknüpfbar.

Eine **Stichprobe** ist eine rein organisatorische Untergruppe (Trägerin von Link und Teilnahmen), die der Gruppenstruktur im Export dient. Das experimentelle Design — Vignetten-Auswahl, Reihenfolge, Randomisierung, Fragebogen-Items — gehört zur **Erhebung**, nicht zur Stichprobe.

## Consequences

- Der Export enthält ausschließlich Teilnahme-Token, keine Identitäten. Etwaige Kontaktdaten liegen getrennt von den Forschungsdaten.
- Trainings können daher niemals als Vorstudie zu einer Erhebung derselben Personen ausgewertet werden. Das ist beabsichtigt.
- Wer den Browser wechselt oder den Speicher löscht, erzeugt eine neue, leere Teilnahme. Der Preis für Anmeldungsfreiheit.
